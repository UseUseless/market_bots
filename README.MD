# Market Bots

Это программа для создания, проверки и запуска торговых роботов. Она позволяет скачать историю цен с биржи, проверить свою торговую стратегию на прошлом (бэктест), подобрать лучшие настройки для прибыли и запустить бота торговать в реальном времени.

Поддерживаются биржи: **Tinkoff (Т-Банк)** и **Bybit**.

## Что умеет программа (Меню Лаунчера)

При запуске программы открывается меню. Вот что делает каждый пункт:

1.  **Управление данными (скачать/обновить)**
    *   Здесь мы скачиваем историю котировок (свечи) с бирж. Без скачанных данных тесты работать не будут. Можно скачать данные по конкретным монетам/акциям или сразу списком.

2.  **Запустить бэктест**
    *   Запуск проверки стратегии на истории. Вы выбираете стратегию, инструмент и период. Программа симулирует торговлю и показывает, сколько денег вы бы заработали или потеряли.
    *   Можно проверить один файл или сразу целую папку с файлами.

3.  **Запустить дэшборд (анализ тестов и боты)**
    *   Открывает в браузере красивую панель с графиками. Там можно детально изучить сделки, посмотреть график роста капитала и сравнить результаты разных стратегий.

4.  **Запустить оптимизацию параметров (WFO)**
    *   Автоматический подбор лучших настроек. Например, программа сама переберет тысячи вариантов стоп-лосса и индикаторов, чтобы найти те, которые дают максимальную прибыль. Используется умный алгоритм Walk-Forward (шагающая оптимизация).
    *   Параметры, шаг, начало и конец перебора параметров изменять в стратегиях и других файлах с параметрами

5.  **Запустить симуляцию в 'песочнице' (Live)**
    *   Запуск бота в режиме реального времени. Он подключается к бирже, следит за ценой и совершает сделки. Пока без счета и денег

## Установка и Запуск (Инструкция для Windows)

Следуйте этим шагам по порядку.

### Шаг 1. Установка Python
Если у вас еще не установлен Python, скачайте версию **3.12** с официального сайта: [python.org](https://www.python.org/downloads/).
*   **Важно:** При установке обязательно поставьте галочку **"Add Python to PATH"**.

### Шаг 2. Подготовка папки
1.  Скачайте этот проект (архивом или через git) и распакуйте его в удобное место.
2.  Допустим, вы распаковали его на диск `D:` в папку `market_bots`.

### Шаг 3. Настройка через консоль
Откройте командную строку (нажмите `Win + R`, введите `cmd` и нажмите Enter). Вводите команды по очереди:

1.  **Переходим в папку с проектом:**
    ```cmd
    D:
    cd D:\market_bots
    ```

2.  **Создаем виртуальное окружение** (это изолированная папка для библиотек, чтобы не засорять систему):
    ```cmd
    python -m venv .venv
    ```

3.  **Активируем окружение:**
    ```cmd
    .venv\Scripts\activate
    ```
    *(После этого в начале строки в консоли должна появиться надпись `(.venv)`)*.

4.  **Устанавливаем необходимые библиотеки:**
    ```cmd
    pip install -r requirements.txt
    ```
    *(Дождитесь окончания установки, это может занять пару минут)*.

### Шаг 4. Настройка ключей (Конфигурация)
Программе нужны ключи от биржи, чтобы скачивать данные.

1.  В папке проекта найдите файл `.env.example`.
2.  Скопируйте его и переименуйте копию в `.env` (просто `.env`, без `.example`).
3.  Откройте файл `.env` любым текстовым редактором (Блокнот, Notepad++).
4.  Вставьте свои токены:
    *   Для **Tinkoff**: Нужен токен "Только для чтения" (для скачивания данных) и "Полный доступ" (если планируете торговать).
    *   Для **Bybit**: Нужны API Key и Secret (для тестов можно оставить пустыми, если скачиваете только публичные данные).
5.  Сохраните файл.

### Шаг 5. Запуск
Теперь всё готово. В той же консоли (где активировано `.venv`) введите:

```cmd
python launcher.py
```
